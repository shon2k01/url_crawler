Frequently Asked Questions

faqs

  • Copilot Mode is a new experimental mode in Microsoft Edge, and our next step towards building a more powerful way to pilot the web. With Copilot Mode on, you enable innovative AI features in Edge that enhance your browser. Copilot Mode is available for free starting today in all Copilot markets, exclusively on Edge for Windows and Mac.* This is a fully opt-in feature for users signed into Edge and will be available for a limited time, so be sure to click and try out the latest features. Help us shape the future of AI-powered browsing. 

  • To turn on Copilot Mode, visit aka.ms/copilot-mode and click the toggle to turn on Copilot Mode in Edge. 

  • It will be available on Windows and Mac worldwide (Copilot markets only) starting on 7/28. 

  • Copilot Mode gives you early access to new AI features that help you get more done while you browse. 

    • Copilot Mode builds on what users love about Copilot in Edgeand adds new ways to chat, take action, and get help right from the browser.  
    • Copilot Actions is also being built directly into Edge so you can hand off tasks for their AI companion to complete on their behalf.  
    • You can also now access multitab context to get higher quality responses from Copilot. 
    • Simple Voice Navigation is also available so you can ask Copilot to navigate to pages as you engage with Copilot Vision. 

  • Copilot Mode is available to all signed-in users in Edge. However, Copilot Pro subscribers have higher usage limits on features like Copilot Actions and Copilot Vision. 

  • If you've opted-into Copilot Mode and want to opt-out, you can do so in Edge Settings.

  • Copilot Mode does not compromise user privacy. Users can choose to turn on Context clues and Personalization in their Settings. Context clues helps Copilot give better answers based on the current webpage, browser history, or browser preferences. While Personalization uses memory from conversations, Bing and MSN activity, and any inferred interests to personalize your experience. If users enroll in Copilot Mode, Copilot will get additional contextual awareness, so it can access additional open tabs to provide higher quality responses, beyond just the tab open on the page. 

  • At launch, Copilot Mode in Edge is available on Windows & Mac. 

  • Copilot Mode is only available to consumers using Edge on Windows and Mac. For the best experience, users should be logged in with the Microsoft account. As we gather feedback and learn more about the experimental features being tested in Copilot Mode we may bring versions of these features to commercial or education customers. 

  • You can join our Copilot in Edge Discord channel: https://discord.com/invite/go-copilot 

  • * Feature availability and functionality may vary by device type, market, and browser version.
