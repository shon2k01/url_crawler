    • Here are the ways you can use Copilot’s AI tools in these social apps:

      On GroupMe

      • Chat with Copilot

      • Generate images

      • Upload images for Copilot to analyze

      On WhatsApp

      • Chat with Copilot

      • Generate images

      • Upload images for Copilot to analyze

      • Send voice messages (Copilot will respond in text format)

      • Get detailed answers using Think Deeper

      On Telegram

      • Chat with Copilot

      • Generate images

      • Upload images for Copilot to analyze

      • Send voice messages (Copilot will respond in text format)

      • Add Copilot in group chats

      • Get detailed answers using Think Deeper

      On Viber

      • Chat with Copilot

      • Generate images

      • Upload images for Copilot to analyze

      • Get detailed answers using Think Deeper

    • Copilot on social apps supports the same languages that Copilot does. See Supported languages for Copilot for more information.

    • On GroupMe:

      • GroupMe does not have a daily message limit.

      On Telegram, Viber, and WhatsApp:

      • You will get 30 turns to chat with Copilot each day.

    • On GroupMe:

      • To remove your chat history stored by Copilot, select the Copilot chat header to return to the main screen. Select the three dots in the upper right corner of the screen, and then select ‘Clear chat history’.

      On Telegram, Viber, and WhatsApp:

      • To remove your chat history stored by Copilot, simply send the message ‘help’ and select ‘delete chat history’.
